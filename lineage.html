This is not a government website. It maps the data flow of an independent tool built on FEMA OpenFEMA, NOAA Storm Events, NOAA/NWS, USGS, and the Data Liberation Project — not endorsed by FEMA, NOAA, NWS, or USGS. Lineage is generated by scripts/build_lineage.py and CI-verified by scripts/verify_lineage.py (the "Guardian"). Freshness is read live from data/manifest.json. Program groupings are a display-only heuristic.
